# Changelog

All notable changes to the "dsaflow" extension will be documented in this file.

## [1.0.1] - 2026-03-18

### Changed
- Hardened the watcher flow with duplicate-event suppression and safer retry handling.
- Added configurable watcher behavior, save prompts, difficulty defaults, and retry command support.
- Switched status bar output to codicons and improved log messaging.
- Added packaging metadata, lint configuration, and a `.vscodeignore` file for Marketplace packaging.
- Rewrote the extension README with install, config, and publishing guidance.

## [1.0.0] - 2026-03-10

### Added
- Initial release of DSAFlow Extension.
- Automatic problem detection and logging.
- AI-powered code analysis integration.
- Spaced repetition revision reminders.
- Secure token storage via `SecretStorage`.
- In-editor Stats WebView.
- Offline queueing for problem logs.
- Commands for login, stats, dashboard, and notes.
